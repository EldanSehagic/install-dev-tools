Automatska skripta za instalaciju i provjeru najnovijih verzija development alata na **Windows 10** koristeći **Git Bash**.

Ova skripta provjerava i instalira sljedeće alate:

- Node.js
- npm
- Express.js (globalno)
- Postman
- WebStorm

---

## ⚡ Kako koristiti

1. Otvori **Git Bash**.
2. Navigiraj do foldera repozitorija `install-dev-tools`:
```bash
cd /c/Users/tvoje_korisnicko_ime/putanja/do/install-dev-tools
```
**Komanda za pokretanje skripte:**
```bash
chmod +x install-dev-tools.sh && ./install-dev-tools.sh
